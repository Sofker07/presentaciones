Función que calcula el factorial de un numero

Esta función toma un número entero como argumento y devuelve su factorial


                            function factorial(n) {
                                if (n === 0 || n === 1) {
                                    return 1;
                                } else {
                                    return n * factorial(n - 1);
                                }
                            }

Función que vencuentra el máximo de una lista de números

Esta función toma una lista de números como argumento y devuelve el número más grande de la lista


                            public static int EncontrarMaximo(List numeros) {
                                int maximo = numeros[0];
                                foreach (int numero in numeros) {
                                    if (numero > maximo) {
                                        maximo = numero;
                                    }
                                }
                                return maximo;
                            }

Bucles

Bucle while

Este bucle ejecuta un bloque de código mientras una condición sea verdadera


                            contador = 0
                            while contador < 5:
                                print(contador)
                                contador += 1

python

Bucle for

Este bucle recorre una secuencia o rango de valores y ejecuta un bloque de código para cada elemento


                            for (let i = 0; i < 5; i++) {
                                console.log(i);
                            }

JavaScript

Bucle do-while

Este bucle ejecuta un bloque de código al menos una vez y luego comprueba si una condición es verdadera para continuar


                            int contador = 0;
                            do {
                                System.out.println(contador);
                                contador++;
                            } while (contador < 5);

Java

Bucle foreach (para cada elemento en una colección)

Este bucle se utiliza para recorrer los elementos de una colección, como una lista o un arreglo


                            numeros = [1, 2, 3, 4, 5]
                            for numero in numeros:
                                print(numero)

python

Bucle for-each en lenguajes modernos

Algunos lenguajes de programación modernos, como Python con su for...in, permiten bucles más simples y legibles


                            nombres = ["Alice", "Bob", "Charlie"]
                            for nombre in nombres:
                                print(nombre)
