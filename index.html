Elementos básicos de la gramática de programación

  • Declaraciones

    • Una declaración determina el nombre y el tipo de datos de una variable u otro elemento

  • Variables

    • Es una forma de almacenar datos que pueden cambiar de valor, de ahí sale su nombre

  • Operadores

    • Es un signo o símbolo que especifica el tipo de cálculo que se realiza en una expresión

  • Palabras clave

    • Son identificadores reservados predefinidos que tienen un significado especial para el compilador

Elementos básicos de la gramática de programación

  • Identificadores

    • Nombres que se proporcionan para variables, tipos, funciones y etiquetas del programa

  • Tipos de Datos

    • Hace referencia al tipo de datos que puede contener y cómo almacena los datos

  • Condicionales

    • Son grupos de sentencias o sentencias individuales que te permiten condicionar la decisión entre la elección de una opción y otra

  • Funcionales

    • También conocidas como subrutinas o procedimientos, son bloques de código a los que se les asigna un nombre

Elementos básicos de la gramática de programación

  • Bucles

    • Los bucles o ciclos de programación se utilizan en los programas de código para establecer sentencias o trozos de código que se repiten o se iteran
