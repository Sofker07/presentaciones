Tipos de Datos

enteros

int

Representan números enteros, positivos o negativos, sin parte decimal


Ejemplos de enteros son 1, -5, y 1000

punto flotante

float

Representan números con parte decimal. Los números de punto flotante pueden ser positivos o negativos


Ejemplos de números de punto flotante son 3.14, -0.5 y 2.0

Caracteres

char

Representan caracteres individuales, como letras, números y símbolos


Por ejemplo, 'a', '1' y '$' son caracteres

cadenas de Caracteres

string

Representan secuencias de caracteres. Las cadenas se utilizan para almacenar texto


Ejemplo: "Hola, mundo!"

booleanos

bool

Representan valores de verdad. Pueden ser verdaderos (true) o falsos (false).


listas o arreglos

list-array

Colecciones ordenadas de elementos del mismo tipo de datos. Los elementos se acceden por índice


Ejemplos: [1, 2, 3] o ["manzana", "banana", "cereza"]

tuplas

tuple

Son similares a las listas, pero son inmutables, lo que significa que no se pueden modificar una vez creadas

Conjuntos

set

Colecciones no ordenadas de elementos únicos. Se utilizan para realizar operaciones de conjuntos, como unión e intersección

Diccionarios

dict

Almacenan pares clave-valor. Cada clave es única y se utiliza para acceder a su valor correspondiente
